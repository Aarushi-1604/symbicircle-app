# SymbiCircle

SymbiCircle is a campus collaboration platform designed to help students discover teammates, share projects, and connect based on technical skills.

The platform allows students to:
- Create profiles with skills and academic details
- Discover collaborators using skill-based search
- Post project ideas or team requirements
- Connect with students across different branches and batches

## Tech Stack

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy

Machine Learning:
- TF-IDF Vectorization
- Cosine Similarity for skill matching

Frontend:
- HTML
- CSS
- JavaScript

Other Tools:
- Git / GitHub
- REST API architecture

## Key Features

- Skill-based collaborator discovery
- Secure authentication system
- Email verification for campus accounts
- Recommendation engine for teammate matching
- Project/team recruitment posts

## Project Goal

The goal of SymbiCircle is to create a centralized platform where students can easily find collaborators for projects, hackathons, and research based on shared skills and interests.
